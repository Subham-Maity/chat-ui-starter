# Basic Chat Ui ⟭

*Easy-to-use chat UI that works with any backend. Makes your app's chat features look good and work well on all devices.*

## ⟭ Starter Project 

### Techs tack
- NextJs
- Tailwind
- Redux
### Installation Guide
> Clone the repo `git clone git@github.com:Subham-Maity/chat-ui-starter.git`
```shell
cd chat-ui-starter

#install
pnpm i 

#run 
pnpm run dev
```
